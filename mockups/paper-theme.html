The Art of Typography

Good typography is invisible. Great typography is a quiet pleasure that readers feel without consciously noticing. The goal is to create a reading experience that feels natural, warm, and unhurried — like words printed on fine paper.

This mockup demonstrates the paper theme for attn, a native markdown viewer. Every element is designed to evoke the tactile quality of printed material.

Heading Hierarchy

A clear typographic hierarchy guides the reader through content without visual noise. Each heading level steps down with intention.

Third-Level Heading

Section separations use generous top margins that increase with heading importance, creating natural breathing room between ideas.

Fourth-Level Heading

At this level, headings feel more like emphasized paragraph leads than section titles.

Tasks

  • Design the color system with warm, paper-inspired tones
  • Select serif typeface for body text
  • Add syntax highlighting with muted, ink-like colors
  • Test across all fixture files

Code Example

Code blocks sit slightly recessed into the page, as if printed on a different stock.

rustfn main() {
    let greeting = "Hello from attn!";
    println!("{greeting}");

    let items = vec!["paper", "ink", "type"];
    for item in &items {
        println!("  → {item}");
    }
}

Inline code like let x = 42 or Vec<String> blends into the paragraph with minimal visual disruption.

Blockquote

Typography is the craft of endowing human language with a durable visual form, and thus with an independent existence.

— Robert Bringhurst, The Elements of Typographic Style

Lists

  • Warm background tones derived from natural paper
  • Ink-like text color — not pure black, but oklch(0.22 0.008 60)
  • Terracotta accent for links and interactive elements
    • Desaturates gracefully in dark mode
    • Maintains WCAG AA contrast
  • Subtle grain overlay simulates paper texture
  1. Choose a warm, neutral base palette
  2. Select a transitional serif for body text
  3. Set generous line-height (1.72) for comfortable reading
  4. Add micro-typography refinements
    1. Negative letter-spacing on headings
    2. Optical margin adjustments

Design Principles

Warmth over neutrality. Every gray has a slight warm bias. Pure blacks and whites are avoided entirely.

Texture over flatness. The subtle grain overlay, inset code block shadows, and paper-toned backgrounds create a sense of physical depth.

Restraint over decoration. The beauty comes from proportion, spacing, and typeface selection — not from ornament.
